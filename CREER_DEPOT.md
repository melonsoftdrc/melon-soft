# Créer le Dépôt sur GitHub

## Étapes pour créer le dépôt

1. **Allez sur GitHub** : https://github.com/new

2. **Remplissez le formulaire** :
   - **Repository name** : `melon-soft`
   - **Description** (optionnel) : `Site web Melon-Soft - Entreprise de développement logiciel en RDC`
   - **Visibility** : Choisissez **Public** ou **Private** selon vos préférences
   - **NE COCHEZ PAS** "Add a README file" (le dépôt local a déjà des fichiers)
   - **NE COCHEZ PAS** "Add .gitignore" (vous en avez déjà un)
   - **NE COCHEZ PAS** "Choose a license"

3. **Cliquez sur "Create repository"**

4. **Une fois le dépôt créé**, revenez ici et je pousserai le code automatiquement.
